About Me

I have over 20 years of experience designing and developing enterprise applications for the Z/OS environment.

I extensively use the CASE tool CA Gen to design and develop CICS server/online transactions and batch applications using DB2 as the DBMS. I also have significant experience designing and developing Z/OS applications using "native" mainframe languages and technologies such as COBOL, JCL, MQ, XML etc.

My experience has been mostly developing applications for the following Australian Commonwealth goverment departments. Here is a brief summary of my most recent projects:

  • Dept of Home Affairs (September 2016 - Present) - The Integrated Cargo System (ICS) is a email/web interfaced application for managing, profiling, calculating duty on cargo. It handles over 250,000 cargo reports and declartions. Its core consists of over 2000 servers and asynchronous triggered transactions executed on the mainframe. These all developed using CA Gen. My role was to help design a new component to maintain Import Permits.
  • Australian Taxation Office (April 2016 - September 2016) The Integrated Core Processing system is a consolidated "single accounting" system that replaces legacy applications handling all income estimation and taxation assessements. It consists of CICS transactions developed using COBOL and using DB2 as the database. My role was to work in a team using the AGILE devlivery methodology to implement changes to the system.
  • Department of Jobs and Small Businesses (formerly Department of Employment Education and Worplace Relations) - - The Employment Services Systems is a web applications that uses server transactions designed and developed using CA Gen. It uses Microsoft's SQL Server as its DBMS
